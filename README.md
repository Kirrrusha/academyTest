Тесты для задач для академии
1. Сортировка
Отсортировать данный массив имён по алфавиту
sortName(['John', 'Jack', 'Jo', 'Kate', 'Low', 'Kirill']) => ['Jack', 'Jo', 'John', 'Kate', 'Kirill', 'Low']
2. Округление
Написать функцию округления чисел, которая принимает 2 параметра, само число и количество знаков после запятой
roudingNumber(10.4566, 2) => 10.46
3. Функция преобразование даты
Написать функцию, которая сравнивает 2 даты, выводит результат в формате "Между 1 сентября 2019г и 3 сентября 2019г - 2 дня", входной параметр дат - "DD.MM.YYYY"
comparisonDates('1.02.2019', '3.08.2019') => Между 1 февраля и 3 августа - 183 дня
4. Склонение слов
Написать функцию, которая склоняет слова, функция принимает 2 параметра число и массив из 3 склонений слова, функция должна вернуть число и слово в корректном склонении
Например declOfNum(10, ['место','места','мест']) => 10 мест
5. Рандомное число в заданном интервале
написать функцию, которая выводит рандомное число в заданном интервале
randomInteger(10, 20) => 15
